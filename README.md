# Machine-Learning-
El Proyecto consta sobre un sistema de inventario creado con HTML, CSS y JS 
conectado con SQL Server de manera local para guardar productos, registros
registros de ingreso y salida de productos, con un sistema de rutas en python.
el entrenamiento de la IA se hizo en Google Colab haciendo primero la carga de
librerias y dataset, luego seentreno usando regresion lineal pero arrojo valores
de MAE y R2 muy bajos, se hizo segundo entrenamiento con RandomFores Regressor y 
los valores mejoraron muy considerablemente.
